E-commerce Project

Project Description

In this project based in e-commerce website like Flipkart , Amazon. I create website to view the product details and add into a cart. And then add to favorite items. There are two sites.Admin and User. Admin can do all work like insert, delete, modify the products, categories and users. User can Register account,login,view the category and choose the product. And add to their cart and favorite item. Without login the page can't do any selection like add to cart and favorite items
Front-end: HTML5, CSS, JAVASCRIPT, BOOTSTRAP 5.
Back-end: PYTHON, MYSQL, DJANGO
